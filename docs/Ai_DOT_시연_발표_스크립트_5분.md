# Ai DOT - 시연 발표 스크립트 (5분 요약본)

| 항목 | 내용 |
|------|------|
| 프로젝트명 | **Ai DOT** (Document Operations & Teamwork) |
| 개발 기간 | 2025.12.05 ~ 2026.02.10 |
| 팀 구성 | 김다흰(총괄/AI), 박은비(프론트), 박제연(백엔드/DB), 이준일(문서/테스트) |
| 발표 시간 | **약 5분** |

---

## 1. 프로젝트 소개 (40초)

> **[화면: 메인 랜딩 페이지 http://localhost:5173]**

안녕하세요. 저희 팀이 개발한 **Ai DOT** 프로젝트를 소개하겠습니다.

Ai DOT는 **AI 기반 통합 업무 협업 플랫폼**입니다.
AI 챗봇, 이미지 생성, 문서 관리, 회의록 분석, 일정 관리를 하나의 플랫폼에서 제공합니다.

가장 큰 특징은 **외부 API 없이 로컬 GPU에서 모든 AI를 처리**한다는 점입니다.
폐쇄망 환경에서도 완전히 독립 운영이 가능합니다.

프론트엔드는 **React 19**, 백엔드는 **FastAPI**,
AI 모델은 **LLaMA 3 한국어**, **Stable Diffusion 3.5**, **Faster Whisper**를 사용했고,
**Master-Worker 분산 아키텍처**로 2대 PC에서 GPU를 분배합니다.

그럼 시연하겠습니다.

> **[조작: 테스트 계정으로 로그인]**

---

## 2. 홈 화면 (20초)

> **[화면: /home]**

홈 화면입니다. 사용자별 **AI 채팅, 문서, 이미지, 일정** 통계가 카드로 표시되고,
아래에 최근 대화와 문서 목록이 있습니다.

> **[조작: 사이드바 "AI 챗봇" 클릭]**

---

## 3. AI 챗봇 — 핵심 기능 (1분 20초)

> **[화면: /chatbot]**

AI 챗봇입니다. **LLaMA 3 Korean Bllossom 8B** 모델이 한국어 대화를 처리합니다.

> **[조작: 새 대화 생성 → 질문 입력 → 전송]**

지금 응답이 **한 글자씩 실시간 스트리밍**으로 나오고 있습니다.
Backend에서 LLM 토큰을 생성할 때마다 **SSE 방식**으로 즉시 전달하는 것입니다.

또한 **RAG 기능**이 있어서, 업로드된 문서를 참조해 더 정확한 답변을 생성합니다.
응답 하단에 참고 문서가 표시되는 것을 확인할 수 있습니다.

> **[조작: 사이드바 "문서보관함" 클릭]**

---

## 4. 문서 보관함 (30초)

> **[화면: /documents]**

문서 보관함입니다. 업로드된 문서를 **카테고리별로 관리**할 수 있습니다.

> **[조작: 카테고리 탭 클릭 → "파일 업로드" 클릭]**

문서를 업로드하면 **백그라운드에서 자동으로 RAG 벡터화가 진행**됩니다.
업로드만 하면 챗봇이 해당 문서를 참조할 수 있게 되는 것입니다.
PDF, DOC, XLS, PPT, HWP 등 다양한 형식을 지원합니다.

> **[조작: 사이드바 "이미지생성" 클릭]**

---

## 5. AI 이미지 생성 — 핵심 기능 (40초)

> **[화면: /images]**

AI 이미지 생성입니다. **Stable Diffusion 3.5 Medium** 모델을 사용합니다.

> **[조작: 한글 프롬프트 입력 → 스타일 선택 → 생성]**

**한글 프롬프트**를 입력하면 LLM이 자동으로 영어로 번역한 뒤,
Worker 서버의 ComfyUI에서 이미지를 생성합니다.
**Celery 비동기 처리**라 생성 중에도 다른 작업이 가능합니다.

> *(미리 생성된 이미지 갤러리 보여주기)*

6종 스타일, 3종 크기를 지원하며, 갤러리에서 상세 확인과 다운로드가 됩니다.

> **[조작: 사이드바 "회의록분석" 클릭]**

---

## 6. 회의록 분석 — 핵심 기능 (40초)

> **[화면: /meeting]**

회의록 분석입니다. 음성 파일을 업로드하면
**Faster Whisper STT**가 텍스트로 변환하고, **LLM이 자동 요약**합니다.

> **[조작: 완료된 회의록 클릭 → 상세 보기]**

이렇게 전체 텍스트 전문과 AI 요약을 확인할 수 있습니다.
직접 작성 모드도 지원합니다.

---

## 7. 관리자 기능 (40초)

> **[조작: 관리자 계정 전환 → /dashboard]**

관리자 대시보드입니다. **CPU, 메모리, 디스크** 실시간 모니터링,
**통계 카드**, **AI 기능 사용 현황**, **시스템 로그**를 한눈에 확인합니다.

> *(사용자 관리/부서 관리 화면 간단히 보여주기)*

사용자 관리에서 **88명의 계정과 역할을 관리**하고,
부서 관리에서 **부서 추가/삭제와 인원 이동**이 가능합니다.

별도로 **AIDot Admin Portal**(Spring Boot)에서
**배포 등록, 라이선스 발급, 다운로드 관리**까지 제공합니다.

---

## 8. 마무리 (30초)

> **[화면: 메인 페이지]**

정리하면, Ai DOT는
**AI 챗봇**(RAG + 스트리밍),
**이미지 생성**(한글 자동 번역),
**회의록 분석**(STT + 요약),
**문서 관리, 일정 관리, 관리자 대시보드, 배포 관리 포탈**을
하나의 플랫폼에서 제공합니다.

모든 AI는 **로컬 GPU**에서 처리되어 폐쇄망에서도 사용 가능하며,
**Master-Worker 분산 구조**와 **Docker Compose 배포**로 확장성과 편의성을 확보했습니다.

감사합니다.
